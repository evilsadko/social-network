# social-network - очередная реализация социальной сети на python

При создании, учитывал возможность использовать в проектах с высокой нагрузкой

#### Для запуска нужно:
* Django - работа с БД
* Tornado - websocket
* Redis - нереляционная высокопроизводительная СУБД
#### Пуск:
***./start***
